About Model Space

Model Space is a 3D visualization tool for exploring scientific models in a three-dimensional space defined by:

  • Concreteness (X-axis)
  • Mathematicality (Y-axis)
  • Computationality (Z-axis)

Each model is represented as a sphere whose:

  • Position reflects its scores on these three dimensions
  • Color is derived from its position (RGB = XYZ)

How to Use

1. Loading Models:

  • Click the "Load Models" button to select a JSON file containing model data
  • The file should contain an array of models with concreteness, mathematicality, and computationality values

2. Navigation:

  • Left-click and drag to rotate the view
  • Right-click and drag to pan
  • Scroll wheel to zoom in/out
  • Hover over models to highlight them

Understanding the Visualization

The three axes represent different aspects of scientific models:

Concreteness (X-axis, Red)

Measures how tangible or physical a model is. Higher values indicate more concrete, physical representations, while lower values suggest more abstract concepts.

Mathematicality (Y-axis, Green)

Represents the degree of mathematical formalization. Higher values indicate more mathematical models with formal equations and proofs, while lower values suggest more qualitative descriptions.

Computationality (Z-axis, Blue)

Indicates how computational or algorithmic a model is. Higher values represent models that are more readily implemented as computer programs or simulations.

Color Coding

Each model's color is determined by its position:

  • Red component comes from Concreteness value
  • Green component comes from Mathematicality value
  • Blue component comes from Computationality value

This creates an intuitive visual connection between a model's position and its appearance.
